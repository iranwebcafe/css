آموزش اچ.ت.ام.ال

سلام، دکتر رضا زعفرانیه هستم.

به وبگاه من خوش آمدید!

زبان اچ.ت.ام.ال یک زبان برای تولید صفحات وب می باشد.

این آموزش اولین مجموعه از سری آموزش های صد روزتا دی.وی.لو.پری است که با استفاده از آن شما قادر خواهید بود در صد روزبه توانایی کامل در زمینه توسعه وب و موبایل تحت دستگاههای Android و ios و windows phone بپردازید.


در این کتاب، اینجانب سایت شخصی خودم را به عنوان مثال

با زبان اچ.ت.ام.ال به صورت قدم به قدم ایجاد

و به شما مفاهیم

زبان اچ.ت.ام.ال را آموزش خواهم داد .

فیلم این کتاب آموزشی و نحوه ساخت این سایت

در ایران وب کافه موجود می باشد

و شما می توانید آن را مشاهده فرمایید .

من در این سری با تهیه فیلم و کتاب به شما همه مهارت های لازم برای تهیه وب سایت ها و موبایل اپ ها را آموزش خواهم داد. برای تهیه این مجموعه با من در تماس باشید.

در مجموعه صد روزتا دی.وی.لو.پری زبانهایی از قبیل :

Html Html5 JavaScript CSS
C# JQuery mobile JQuery delphi
php asp.net java Json



و پلتفرم های لازم برای توسعه برنامه های موبایلی و وبی از قبیل:

Visual Studio for microsoft Rad Studio for Embarcadero BlueMix for IBM
PhoenGap Android Studio for Google ADF for Oracle


را آموزش خواهید دید و به توانایی کامل در این زمینه خواهید رسید .
